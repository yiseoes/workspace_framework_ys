# workspace_framework_ys

Spring Framework 학습 기록입니다.

Java 기반 웹 개발을 공부하며 작성한 프로젝트들을 모아두었습니다.

---

## Spring Core 학습

| 프로젝트 | 학습 내용 |
|---------|----------|
| `01base` | Spring 기초 개념 |
| `02ioC Container` | IoC(Inversion of Control) 컨테이너, 의존성 주입(DI) |
| `03AOP` | AOP(Aspect Oriented Programming), 관점 지향 프로그래밍 |
| `04SpringMVC` | Spring MVC 패턴 기초 |
| `05SpringAnnotatinoMVC` | 어노테이션 기반 Spring MVC |
| `06SpringWeb` | Spring 웹 애플리케이션 구성 |

---

## Model2MVCShop 시리즈

쇼핑몰 프로젝트를 단계별로 발전시키며 학습한 기록입니다.

| 프로젝트 | 학습 내용 |
|---------|----------|
| `03.Model2MVCShop(EL,JSTL)` | EL(Expression Language), JSTL 활용 |
| `04.Model2MVCShop(Business Logic,MyBatis Spring)` | 비즈니스 로직 분리, MyBatis-Spring 연동 |
| `05.Model2MVCShop(AOP,Transaction)` | AOP 적용, 트랜잭션 처리 |
| `06.Model2MVCShop(Presentation+BusinessLogic)` | 프레젠테이션/비즈니스 레이어 통합 |
| `07.Model2MVCShop(URI,pattern)` | RESTful URI 패턴 설계 |
| `08.Model2MVCShop(RestFul Server)` | REST API 서버 구축 |
| `08.Model2MVCShop(RestFul Client)` | REST API 클라이언트 구현 |
| `09.Model2MVCShop(jQuery)` | jQuery를 활용한 프론트엔드 개선 |
| `10.Model2MVCShop(Ajax)` | Ajax 비동기 통신, OAuth 2.0 소셜 로그인(Kakao, Google, Naver) |
